# Vote-for-What-app

## About The Project
![screenshot](https://www.gabe-dev.com/images/vfw.png)

Allows users to easily search for ballot initiatives based on location and keywords, and set up notifications on future ballots matching their search.




### Built With

* [HTML]
* [CSS]
* [SCSS]
* [EJS]
* [Javascript]
* [Express.js]
* [Node.js]
* [Bootstrap]
